# PolyBool2D 操作マニュアル

PolyBool2D は、建築設計などで利用する **2D ポリゴン CAD Web アプリ** です。
ポリゴンの作図・編集・ブーリアン演算・ナイフ分割を行い、面積を算定できます。

---

## 1. 画面構成

- **ヘッダー**: プロジェクト操作（新規 / インポート / エクスポート）、言語・テーマ切替、マニュアル
- **ツールバー**: 作図／編集ツールの切替
- **キャンバス**: SVG 描画領域。マウスホイールでズーム、スペース + ドラッグまたはパンツールで画面移動
- **プロパティパネル**: 選択図形の情報、座標一覧、面積、設定
- **ステータスバー**: カーソル座標、操作ガイド

---

## 2. 基本操作

| 操作 | 内容 |
| --- | --- |
| `V` | 選択ツール |
| `H` | パンツール |
| `P` | ポリゴン作図 |
| `R` | 矩形作図 |
| `C` | 円作図 |
| `K` | ナイフ |
| `Enter` | 作図確定 |
| `Esc` | 作図キャンセル |
| `Delete` / `Backspace` | 選択削除 |
| `Ctrl/⌘ + Z` | 元に戻す |
| `Ctrl/⌘ + Shift + Z` | やり直し |
| `G` | グリッド ON/OFF |
| `S` | スナップ ON/OFF |

---

## 3. 作図

### ポリゴン
1. ツールバーから「ポリゴン」を選ぶ。
2. キャンバスをクリックして頂点を追加。
3. `Enter` で確定（3 点以上必要）、`Esc` で取消。

### 矩形
1. ツールバーから「矩形」を選ぶ。
2. ドラッグで対角線を指定。`Shift` で正方形。

### 円
1. ツールバーから「円」を選ぶ。
2. 中心からドラッグで半径を指定。
3. 円は内部的に多角形（既定 64 分割）として保存されます。

---

## 4. 頂点編集

選択ツールで図形を選ぶと、頂点ハンドルが表示されます。

- ハンドルをドラッグで頂点を移動
- プロパティパネルの座標表で外周頂点の X/Y を直接入力可能
- 現バージョンでは頂点の追加・削除は未実装です

---

## 5. ブーリアン演算

### 結合（Union）
2 つ以上の図形を選択 → ツールバー「結合」 をクリック。重なりが統合されます。

### くり抜き（Difference）
1. 対象図形を最初に選択。
2. `Shift` クリックでカッター図形を追加選択。
3. ツールバー「くり抜き」 をクリック。

例: 矩形を最初に選び、円を `Shift` クリック → くり抜きで円形の穴があきます。

---

## 6. ナイフ分割

1. 分割したいポリゴンを選択。
2. ツールバー「ナイフ」を選ぶ。
3. 図形の外周をちょうど 2 点で横切る直線をドラッグ。
4. ドラッグを離すと分割を実行します。分割前後の面積合計は一致します。

> 現バージョンは **単純ポリゴン × 1 直線 × 2 交点** のみ対応します。

---

## 7. 面積

- ポリゴン外周面積から穴面積を差し引いた **実面積** を表示
- 既定の座標単位は mm。面積は m² で表示されます
- 全選択合計、全体合計をプロパティパネルに表示

---

## 8. 保存・出力

- ローカル自動保存（`localStorage`）
- ヘッダー「エクスポート」で JSON ダウンロード
- 「インポート」で JSON 復元

---

## 9. 既知の制限

- ナイフは 1 直線・2 交点の MVP 実装
- 自己交差ポリゴンはサポート外
- 頂点の追加・削除は未実装
- 円は多角形近似
- 3D / BIM 連携は対象外
